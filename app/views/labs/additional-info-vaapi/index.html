1. Insert the USB drive to your NUC and copy the top level folder "hdc_2.2_conf_files" to /root folder and go to that folder

cd /root
cp -r /media/sda1/HDC/hdc_2.2_conf_files /root
cd /root/hdc_2.2_conf_files

2. Run the shell script

./hdc_2.2.0_install.sh

3. Wait for the script to complete, at the end it will reboot the gateway. This step will take upto 5 minutes.
