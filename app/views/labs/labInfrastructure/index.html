  4. Lab Infrastructure
  1. Unscrew the NUC kit.
  2. Upgrade 1 DDR3 RAM in provided slot properly
  3. Insert Solid state drive in the back case.
  4. Screw the NUC kit.
  5. Power on the NUC with Monitor connected.
  6. Press F10 and boot from Windows 10 USB.
  7. Install Windows 10.

  8. Set Username: intel<NUC Number>

    and Password: intel123

  9. Open the command prompt in admin mode and navigate to Workshop folder in DATA USB
  10. Goto 00-Installers, RunMe1.bat to install drivers.
  11. Reboot the system.

  12. Execute RunMe2.bat to install WindowsSDK, Directx 11 and Visual Studio 10.

    Note:- Directx installation may ask for .NET framework 3.5 for which internet connection is a must.

  13. Execute RunMe3.bat to unmount the VS image, register Mars dlls, install opencv and Mesh commander.
  14. Copy folders 01 - 04 to Desktop > Retail
  15. Through elevated command prompt run Desktop > Retail > 01-MARS > filterDlls > regdll.bat, execute to register the dlls required for Mars
  16. Goto Desktop Retail > 01-Mars > Bin > Right Click on MARS.exe and Send to Desktop (create a link).
  17. Run MARS.exe and check if one video is running properly.
  18. Run Python idle. Type: import cv2

    It should not give any error.

    Type: print cv2.__version__

    It should display the cv2 version.

  19. Create System Restore point named "Retail".
  20. Shutdown the system, unplug the cables and stick Intel logo provided in the box, on to the NUC and pack the box.
